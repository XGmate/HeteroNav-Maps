# HeteroNav-Maps
#HeteroNav‑Maps: A Benchmark for Spatially Adaptive Path Planning

This dataset consists of composite binary occupancy maps, each integrating multiple geometric structures within a single environment, including open areas, narrow passages, skeleton‑dominant corridors, and dead‑end regions. These heterogeneous layouts are specifically designed to evaluate the spatial adaptability of path planning algorithms, particularly those that rely on multimodal morphological features and adaptive fusion mechanisms. Each map is provided with multiple start‑goal pairs, and all maps are formatted as 0/1 matrices with consistent resolution.
